## `regional_data.py`
Fetches completed and active regionals for the current season, creates file in `logs/YEAR/event_code.json` of match data.
**You will need to create an Account with TBA, create an API key, and place in in the root directory inside a `tba_key.txt` file**

## `auto_vs_tele.py`
Compares correlation between auto win and match win

## `climb_rp.py`
Fetches matches where at least one alliance has been awarded the climb rp

## `match_delta`
Uses matplotlib to plot a graph of the winning deltas for all matches

## `uncounted_fuel.py`
Displays fuel scoring stats, such as total fuel scored, periods scored in, and total count of uncounted fuel (i.e. fuel scored not during alliances periods)
